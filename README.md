_Mean Mobs makes animals act more like real animals in Minecraft._

**Features:**
- If you get too close to a sheep/cow/pig/chicken they will flee
- If you punch a sheep/cow/pig they have a chance of biting back
- Animals will seek cover during rain
- Optimized
- More features planned!
